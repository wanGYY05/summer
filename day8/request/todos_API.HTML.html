TODOS API 文档

注释:

1.初始账号是你们自己的名字，密码是123456.

2.以下英文:url代表请求的路径, method代表请求的的方式(常用的有get,post,delete,put,patch), ,filed代表请求的时候所带的字段。

3.根据发送的请求服务端返回的结果进行相应的操作，比如登录的时候服务端返回： 

    {
        code:10000,
        data:{
            msg:'登录成功'，
            id:'xxxxxxxxxxx'
        }
    }

代表登录成功了，这个时候可以让用户进入到主页，并且将获取到的id存到localStorage中，提供给后面获取todo列表等使用，相当于是你的令牌.

4.如果url里面出现了/:id,这个url里面的这个是一个未知数，学术名称叫做parmas，在进行请求的时候把这个替换成自己登录成功获取到的 id即可。

1. 登录

    {
        url:http://api.neuqstlab.qoder.cn/users/login
        method:POST,
        fileds:{
            name:String,
            password:String
        }
    }


例： 

    fetch('http://api.neuqstlab.qoder.cn/users/login',{
         method: 'POST',
         headers: {
            'Accept': 'application/json',
            'Content-Type': 'application/json'
         },
         body: JSON.stringify({
         name: 'test',
         password: '123456'
        })
    }).then(function(res){
         return res.json();
    }).then(function(json){
        console.log(json);
    })

2.获取todos列表

    {
        url:http://api.neuqstlab.qoder.cn/users/todos/:id
        method:GET,
    }

3.添加一条todo

    {
        url:http://api.neuqstlab.qoder.cn/users/todos/:id
        method:POST,
        fileds:{
            title:String,
            time:String,
            content:String
        }
    }

4.清空todo列表

    {
        url:http://api.neuqstlab.qoder.cn/users/todos/:id
        method:DELETE
    }
